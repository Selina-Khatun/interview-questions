1. What is HTML?

  • HTML stands for HyperText Markup Language and is the language of the internet. It is the standard text formatting language used for creating and displaying pages on the Internet HTML documents are made up of the elements and the tags that format it for proper display on pages. (এইচটিএমএল হল হাইপারটেক্সট মার্কআপ ল্যাঙ্গুয়েজ এবং এটি ইন্টারনেটের ভাষা। এটি ইন্টারনেটে পৃষ্ঠাগুলি তৈরি এবং প্রদর্শনের জন্য ব্যবহৃত স্ট্যান্ডার্ড টেক্সট ফর্ম্যাটিং ভাষা HTML নথিগুলি উপাদান এবং ট্যাগগুলি দিয়ে তৈরি যা পৃষ্ঠাগুলিতে সঠিক প্রদর্শনের জন্য এটিকে ফর্ম্যাট করে।)

  • HTML is used to design web pages using a markup language. HTML is a combination of Hypertext and Markup language. Hypertext defines the link between the web pages. The markup language is used to define the text document within the tag which defines the structure of web pages. HTML is used to structure the website and is therefore used for Web Development. (HTML একটি মার্কআপ ভাষা ব্যবহার করে ওয়েব পেজ ডিজাইন করতে ব্যবহৃত হয়। এইচটিএমএল হল হাইপারটেক্সট এবং এর সংমিশ্রণ মার্কআপ ভাষা. হাইপারটেক্সট ওয়েব পৃষ্ঠাগুলির মধ্যে লিঙ্ক সংজ্ঞায়িত করে। মার্কআপ ভাষা ব্যবহার করা হয় ট্যাগের মধ্যে টেক্সট ডকুমেন্ট সংজ্ঞায়িত করতে যা ওয়েব পৃষ্ঠাগুলির গঠন সংজ্ঞায়িত করে। এইচটিএমএল হল ওয়েবসাইট গঠন করতে ব্যবহৃত হয় এবং তাই ওয়েব ডেভেলপমেন্টের জন্য ব্যবহৃত হয়।)

  • HTML is the language of the World Wide Web. It is the standard text formatting language used for creating and displaying pages on the Web. HTML documents are made up of two things: the content and the tags that format it for proper display on pages.(HTML হল ওয়ার্ল্ড ওয়াইড ওয়েবের ভাষা। এটির জন্য ব্যবহৃত স্ট্যান্ডার্ড টেক্সট ফরম্যাটিং ভাষা ওয়েবে পৃষ্ঠা তৈরি এবং প্রদর্শন করা। HTML নথি দুটি জিনিস নিয়ে গঠিত: বিষয়বস্তু এবং ট্যাগগুলি যা পৃষ্ঠাগুলিতে সঠিক প্রদর্শনের জন্য এটিকে ফর্ম্যাট করে।)

  • HTML is a standard text formatting language which is used to create and display pages on the Web. HTML makes the text more interactive and dynamic. It can turn text into images, tables, links. HTML pages are saved by adding .html or .html in web page name.(এইচটিএমএল হল একটি স্ট্যান্ডার্ড টেক্সট ফরম্যাটিং ল্যাঙ্গুয়েজ ওয়েবে পৃষ্ঠাগুলি তৈরি এবং প্রদর্শন করতে ব্যবহৃত হয়। এইচটিএমএল টেক্সটকে আরও ইন্টারেক্টিভ এবং গতিশীল করে তোলে। এটি পাঠ্যকে চিত্র, টেবিল, লিঙ্কে পরিণত করতে পারে। HTML পৃষ্ঠাগুলি .html বা .html যোগ করে সংরক্ষণ করা হয় ওয়েব পৃষ্ঠার নাম।)

  • HTML or Hypertext Markup Language, was created by Berners-Lee in 1991. It is a markup language used to create and structure website templates or web pages to present content on the World Wide Web. It consists of a series of elements, and the HTML elements tell the browser how to display the content. HTML helps in making the text more interactive and dynamic. You can save an HTML page by adding .html or .html in the web page name.(এইচটিএমএল বা হাইপারটেক্সট মার্কআপ ল্যাঙ্গুয়েজ, বার্নার্স-লি 1991 সালে তৈরি করেছিলেন। এটি একটি মার্কআপ ভাষা বিশ্বব্যাপী বিষয়বস্তু উপস্থাপন করার জন্য ওয়েবসাইট টেমপ্লেট বা ওয়েব পৃষ্ঠাগুলি তৈরি এবং গঠন করতে ব্যবহৃত হয় ওয়েব এটি উপাদানগুলির একটি সিরিজ নিয়ে গঠিত, এবং HTML উপাদানগুলি ব্রাউজারকে কীভাবে প্রদর্শন করতে হবে তা বলে সূচিপত্র. এইচটিএমএল পাঠ্যকে আরও ইন্টারেক্টিভ এবং গতিশীল করতে সহায়তা করে। আপনি একটি HTML সংরক্ষণ করতে পারেন ওয়েব পেজের নামের সাথে .html বা .html যোগ করে পৃষ্ঠা।)
